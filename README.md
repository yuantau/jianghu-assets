# Jianghu Assets

提供[Jianghu](https://github.com/yuantau/1000y)所需的资源文件，仅供学习交流使用， 如有侵权请联系删除。

资源提取自网络游戏[千年](https://baike.baidu.com/item/%E5%8D%83%E5%B9%B4/9011313#viewPageContent), 该游戏在 2006 年于大陆地区停止运营，目前续作[千年 3](https://1000y.web.sdo.com/web4/)依旧在运营。

## 使用

- 从`Releases`列表下载最新的 zip 文件解压到相应目录。

- 为`Game Tools`,`Map Editor`指定上一步骤的解压路径。
